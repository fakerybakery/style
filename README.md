# Style
### My style of coding
* Use `'` single quotes instead of `"` double quotes unless you really need to!
* Don't put spaces between `(` and the statement (e.g. `if ($a == $b)` rather than `if ( $a == $b )`
* Use `background` over `background-color` or `background-image`
